Learn to Code!

Want to learn how to code? Look at these websites for some help.

w3schools

w3schools is helpful for much of the basic html and css. Almost everything is there from images, links, to font, and borders. It has helpful tips and the code is already there. Great for if you're a beginner.

jQuery

jQuery is a place to write javascript and create small, quick websites. Good if you're experimenting with fonts, colors, anything you're unsure of.

GitHub

GitHub is what you're viewing this website on! GitHub is a free website where you can write code and create a website like this!! (You should really try it, it's pretty cool.)
